
# Wanted 공식 홈페이지 클론 프로젝트 
-----------------------------------
## 🖥 프로젝트 소개


* 주제: 원티드(http://www.wanted.com) 클론 코딩 프로젝트입니다.
* 프로젝트 기간: 2020.10.05 ~ 2020.10.16
* Git, Trello, Slack을 이용한 협업

## 🛠 기술 스택

* Front-end
  - HTML, CSS
  - JavaScript(ES6+)
  - React (CRA, Hooks, Redux)
  - Slick.js (library)
  - HighChart.js
  - Google MAP API
  - Styled Component
  
* Back-end
  - Pyhon, Django
  - MySQL
  - Bcrypt, PyJWT
  - CORS headers
  - RESTful API
  - Boto3(AW-S3 Bucket 연동 라이브러리)
